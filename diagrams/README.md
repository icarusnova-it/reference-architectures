# Architecture Diagrams

This directory contains C4 model diagrams for all reference architectures.

## Diagram Types

### C4 Model

We use the C4 model for architecture diagrams:
- **System Context**: High-level view showing system and external actors
- **Container**: Major components and their interactions
- **Component**: Internal structure of containers (where applicable)

## Diagram Standards

- **Format**: Mermaid diagrams in Markdown
- **Style**: Consistent color coding and layout
- **Detail Level**: Container-level view (high-level)
- **Annotations**: Clear labels and descriptions
